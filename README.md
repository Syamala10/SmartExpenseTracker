# 💰 Smart Expense Tracker

A Java Web Application developed using JSP, Servlets, JDBC, MySQL, Bootstrap, and Apache Tomcat.

## 📌 Features

- User Login Authentication
- Session Management
- Add Expense
- View Expenses
- Update Expense
- Delete Expense
- Logout

## 🛠 Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- Bootstrap 5
- Apache Tomcat 10
- Eclipse IDE

## 🗄 Database

Database Name:

smart_expense_tracker

## 📂 Project Architecture

MVC Architecture

- Model
- DAO
- Controller (Servlets)
- JSP Views
